      Aviso operacional

      Antes da operação comercial, revise requisitos legais, fiscais e regulatórios aplicáveis ao transporte remunerado privado no município.

      Aplicativo Android

      O front-end pode ser executado no Android via Capacitor, usando API local ou API online com HTTPS.

      Modo atual: API local/mesmo domínio.

      Permissões do aplicativo

      Verifique e solicite permissões para garantir localização e recursos no Android.

      As permissões agora aparecem diretamente na primeira tela de login.

      • Front-end dentro da pasta public.
      • Banco SQLite em data/pardogo.sqlite.
      • Mapa com Leaflet/OpenStreetMap no navegador.
      • Cálculo de rota via API do backend e fallback por coordenadas.
      • Tempo real com SSE/EventSource para passageiro, motorista e admin.
      • Cancelamento de corrida com motivo e auditoria.
      • Contato entre passageiro e motorista por WhatsApp/ligação.
      • Avaliação da corrida com nota de 1 a 5 e comentário.
      • Média de avaliação do motorista e alertas de qualidade no admin.
      • Suporte, denúncia, termos, privacidade e revisão documental do motorista.
      • Base Android com Capacitor documentada no arquivo APP.md.
      • Configuração de API online por public/mobile-config.js ou pela tela acima.
      • Scripts npm run cap:add:android, npm run cap:sync:android e npm run cap:open:android.
